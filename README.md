Guía rápida de UI
- Tipografías: `--font-display` para títulos y cifras, `--font-sans` para texto.
- Elevaciones: usa `--elev-1/2/3` y `--elev-press` para sombras coherentes.
- Botones: tamaños estándar `--btn-height`, `--btn-height-lg`, `--btn-height-sm` y radios `--btn-radius*`.
- Contraste: `--muted` pensado para textos secundarios sobre `--bg`.
- Animaciones: respeta `prefers-reduced-motion`.
